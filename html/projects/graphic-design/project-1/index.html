Twisted Hair Brushes

Type

Graphic Design

Date

31.10.2024

Scale

148 x 210 mm (Edition)

Scale 2

420 x 594 mm (Poster)

Know-How

Poster Design, Editorial Design,

This mini-edition features six distorted combs illustrated on each page. Through deliberate alteration, these everyday objects are transformed into sculptural forms that transcend their purely utilitarian purpose. And with those elements, we had to create a poster, that represents the design of this project.
