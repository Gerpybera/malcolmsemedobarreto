Clôture Fête fédérale de gymnastique Lausanne 2025

Type

Motion Design

Date

21.06.2025

Scale

29460 x 90 px

Know-How

Mouvement Image, 3D Graphics

For this collaborative project, I created 11 banners, each representing a group participating in the Lausanne gymnastics festival. Each group adopts a color from the rainbow. In each animation, I created the 3D visuals and animated the movement of logos and pictograms.
