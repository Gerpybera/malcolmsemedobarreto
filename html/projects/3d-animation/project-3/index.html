METAMORPHOSIS

Type

3D Animation

Date

06.01.2025

Scale

1920 x 1080 px

Know-How

Mouvement Image, 3D Graphics

Exploring the theme of "metamorphosis," this work traces a transformation from a distorted barrel to a pristine diamond. Each visual contrast represents a stage in this evolution, highlighting the mutations of matter and form as they shift from industrial decay to crystalline perfection.
