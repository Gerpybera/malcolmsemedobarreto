Homunculus Lab

Type

Media & Interaction Design

Date

11.04.2025

Scale

1920 x 1080 px

Know-How

UI/UX design, Web, Creative Coding, Three.js

Homunculus Lab is an interactive web project that allows users to create a character from collages of figurative and abstract objects. Users compose a unique, downloadable avatar by exploring infinite compositional possibilities. This project explores the theme "hypermetamorphosis". The body is deconstructed and reinvented using only elements with no direct link to human anatomy, creating unrecognizable and surreal characters that translate an extreme transformation of visual identity. Users can animate their avatar, reinforcing the living dimension of this digital creature. This approach questions: what is an avatar? Can it reflect personal identity even when departing from the human figure? Inspired by Victoria Vesna's Bodies INC and Giorgio De Chirico's universe, Homunculus Lab pushes customization toward a surrealist aesthetic where each object choice becomes an act of personal expression.
