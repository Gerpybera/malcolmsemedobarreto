Invisible Layer - Monetization

Type

Media & Interaction Design

Date

19.12.2024

Scale

1920 x 1080 px

Know-How

Creative Coding, P5.JS

This exercise involved designing a Chrome extension that reveals invisible data patterns. My extension exposes hidden monetization data from websites by dividing pages into sections that correspond to the number of advertisements present. Clicking on these sections reveals the revenue generated by each ad placement. Mouse movement across the site dynamically alters its structure, symbolizing the act of obfuscating information flow.
